Key locations in the project

- App entry: `lib/main.dart`
- Screens: `lib/screens/` (Home, Wastec Bank, Wallet, Trending, Track Order, etc.)
- Shared widgets: `lib/widgets/` (e.g., `profile_wallet_actions.dart`, `wastec_bank_category_bar.dart`)
- Data & controllers: `lib/data/`, `lib/controllers/`
- Build outputs: `build/` and `builds/` (packaged APK/ZIP created during release)
- Docs: `DOCS/` (this folder)

If you are new to the project, start with `DOCS/INSTALLATION.md` and `DOCS/DEVELOPMENT.md`.
